YouTrek is a lightweight macOS app that connects to YouTrack so you can browse, search, and manage issues without living in a browser tab.

SwiftUI app with platform-native menus, windows, and shortcuts.

OAuth login via YouTrack and tokens stored in macOS Keychain.

Open source

Built in public—browse the code, file issues, and contribute.
